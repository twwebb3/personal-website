RStudio

I've been using RStudio since 2013, it's the only way to use R. When I first started using R in 2012 my default was to create a folder and setwd() to that folder, but it's much better practice to create a project in RStudio and completely dispense with setwd(). Everything is so much more seamless in with R in RStudio: plots, seeing your directory, viewing data, and lastly developing Shiny apps.
